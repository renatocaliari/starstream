# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0-alpha.1] - 2025-03-07

### Added
- Initial alpha release
- Core broadcasting engine
- Presence system
- Typing indicators
- Cursor tracking
- Message history
- SQLite storage backend
- PocketBase plugin
- Loro CRDT plugin
- 124 test cases

### Known Issues
- Alpha quality, API may change
- Limited documentation
- Some edge cases not covered

## [0.1.1] - 2026-03-07

### Added
- **AI Agent Skill**: New `SKILL.md` following 2026 best practices for building real-time apps.
- **SQLite Defaults**: Documented that SQLite is the default storage backend for zero-config persistence.

### Changed
- Improved `README.md` with explicit storage instructions.
- Refined monorepo structure documentation.
